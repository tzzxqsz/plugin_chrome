隐私政策 (Privacy Policy)

最近更新日期：2026年6月1日

本隐私政策适用于您使用“代码计数工具”（Code Counter）Chrome 浏览器扩展程序（以下简称“本插件”）。我们高度重视您的隐私，并承诺保护您的代码安全。

1. 数据收集与使用

我们不收集任何个人数据或代码数据。

  • 本地处理： 当您选择文件或文件夹进行代码计数时，所有文件的读取、解析和统计操作完全在您的本地浏览器中进行。
  • 无服务器传输： 本插件不设任何后端服务器，绝不会将您的代码、文件名、项目结构或任何其他数据上传到互联网。

2. 权限说明

本插件仅申请运行所必需的最低权限。所有权限仅用于在本地读取您主动上传/选择的代码文件，绝不会用于跟踪您的浏览行为。

3. Cookie 与第三方服务

本插件不使用任何 Cookie，不包含任何第三方广告追踪、分析代码或统计 SDK。

4. 变更通知

如果我们未来对本插件的隐私政策进行修改，我们将通过更新本页面向您通知。但在任何情况下，我们都不会改变“不收集用户代码”的核心原则。

5. 联系我们

如果您对本隐私政策有任何疑问，请通过您的应用商店反馈渠道联系我们。

Privacy Policy

Last updated: June 1, 2026

This Privacy Policy applies to your use of the "Code Counter" Chrome extension (the "Extension"). We highly value your privacy and are committed to protecting your source code.

1. Data Collection and Usage

We do not collect any personal data or source code.

  • Local Processing: When you select files or folders for counting, all file reading, parsing, and analysis are executed entirely within your local browser environment.
  • No Data Transmission: The Extension has no backend server and will never upload your code, file names, project structure, or any other data to the internet.

2. Permissions

The Extension only requests the minimum permissions required for local file analysis. These permissions are never used to track your browsing habits or collect history.

3. Cookies and Third-Party Services

The Extension does not use cookies, and it does not include any third-party advertisements, tracking scripts, or analytics SDKs.

4. Contact Us

If you have any questions regarding this Privacy Policy, please contact us via the Chrome Web Store support/developer channel.
