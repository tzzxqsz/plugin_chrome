Privacy Policy

This extension does not collect, store, or share any personal user data.

Data Usage

  • No personal information is collected
  • No tracking or analytics
  • No cookies are used

External APIs

This extension uses third-party weather APIs (such as Open-Meteo) to fetch weather data. These requests do not include any personally identifiable information.

Storage

The extension uses Chrome local storage only to save user-added cities locally on the user's device.
