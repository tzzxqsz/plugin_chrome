JWT Decoder 隐私权政策

本隐私权政策阐明了 JWT Decoder（以下简称“本扩展程序”）如何处理用户的数据。我们非常重视您的隐私，并承诺不收集、不存储、不分享您的任何个人信息或输入的数据。

1. 数据收集与使用

本扩展程序是一款纯本地运行的开发者工具：

  • 不收集数据： 本扩展程序不会收集、记录、读取或存储您的任何个人身份信息、浏览历史或您在输入框中粘贴的任何 JWT Token 数据。
  • 纯本地解析： 您输入的所有 JWT Token 均在您的本地浏览器（客户端）中进行解析和处理，整个解码过程完全离线完成。
  • 无网络传输： 本扩展程序没有配置任何网络权限（未申请任何网络请求 API），绝不会向任何外部服务器或第三方传输您的任何输入内容。

2. 权限使用说明

为了提供核心功能，本扩展程序申请了以下特定权限：

  • clipboardWrite（剪贴板写入权限）： 仅用于当用户点击界面上的“Copy”按钮时，将格式化后的 JSON 文本写入系统剪贴板。我们绝对不会读取您的剪贴板内容。

3. 第三方服务与政策变更

本扩展程序不包含任何第三方广告、追踪代码或分析组件。如果未来由于功能升级导致本隐私政策发生变更，我们会在 Chrome Web Store 的更新说明中明确提示用户，并更新本页面。

4. 联系我们

如果您对本隐私政策或本扩展程序有任何疑问，请通过 Chrome Web Store 开发者支持页面或通过支持邮箱与我们取得联系。
