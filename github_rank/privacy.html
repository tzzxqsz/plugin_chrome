GitHub Rank 扩展程序隐私政策

生效日期： 2026年6月1日

我们非常重视您的隐私。本隐私政策旨在明确阐述“GitHub Rank”浏览器扩展程序（以下简称“本扩展”）在您使用过程中的数据处理情况。

1. 数据收集与使用说明

本扩展承诺不收集用户的任何个人数据。

  • 个人身份信息：本扩展不需要您注册、登录，亦不会收集您的姓名、电子邮箱、GitHub 账号等任何个人身份信息。
  • 浏览历史与行为痕迹：本扩展没有后台常驻脚本（Background Scripts），无法也绝不追踪、读取、存储、记录您的任何网页浏览历史、网络活动痕迹。

2. 网络请求与外部通信

为了向您展示最新的 GitHub 项目排行，当您主动点击扩展图标打开弹出窗口（Popup）时，本扩展会向我们托管在 Cloudflare 上的数据服务器（https://github.9191ta.com）发起不带任何标识符的、匿名的数据读取请求（GET）。该请求仅用于下载公共开源排行榜数据，不包含任何可以识别您设备或身份的信息。

3. 隐私政策的变更

我们可能会适时对本隐私政策进行修订。若有任何变更，我们会在本页面发布更新版本。继续使用本扩展即代表您同意最新的隐私政策。

如有任何疑问，请通过 GitHub 仓库 Issue 页面或开发者邮箱与我们联系。

Privacy Policy for GitHub Rank Extension

Effective Date: June 1, 2026

We highly value your privacy. This Privacy Policy clarifies how "GitHub Rank" (the "Extension") handles information when you install and use our browser extension.

1. Information Collection and Use

The Extension DOES NOT collect any of your personal data.

  • Personal Identity Information: The Extension does not require account registration or login. It never collects, stores, or transmits your name, email address, GitHub username, or any identifiers.
  • Browsing History & Activity: The Extension does not contain persistent background service workers or content scripts. It cannot and will never track, read, or intercept your browsing history or online activities.

2. Network Communications

To display up-to-date GitHub trending leaderboards, the Extension sends an anonymous HTTP GET request to our secure data endpoint (https://github.9191ta.com) only when you explicitly click the extension icon to open the popup. This request carries no authentication tracking parameters, user data, or device fingerprints; it is solely used to download the public JSON list of open-source repositories.

3. Changes to This Privacy Policy

We may update our Privacy Policy from time to time. Any changes will be posted immediately on this page. Your continued use of the Extension after changes constitutes acceptance of the revised policy.

If you have any questions, feel free to contact us via the GitHub project issue tracker.
