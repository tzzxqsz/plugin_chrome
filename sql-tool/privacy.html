Privacy Policy

Effective Date: May 24, 2026

This Privacy Policy governs the data practices of the SQL Tool Formatter Chrome Extension (the "Extension"). We are committed to protecting user privacy and ensuring a secure environment.

Core Principle: 100% Offline & Private
This Extension is a pure client-side utility. All SQL formatting, minification, and processing logic are executed entirely inside your local browser. No data leaves your machine.

1. Data Collection and Usage

We DO NOT collect, store, transmit, or share any personal data, web browsing history, or the SQL code you input into the Extension.

  • SQL Queries: Any code you paste, select, or format is handled exclusively in the local volatile memory and is destroyed when the tab or popup closes.
  • No Analytics: We do not use any third-party tracking scripts, telemetry, or analytics platforms (e.g., Google Analytics).

2. Explanation of Permissions Used

The Extension requests the following permissions solely to enable core features locally:

  • contextMenus: Allows you to select SQL text on a webpage, right-click, and format it instantly.
  • storage: Used strictly to remember your local settings (e.g., light/dark mode, indentation preferences) and optional local formatting history inside your own browser.
  • activeTab & Host Permissions (<all_urls>): Necessary to allow the context menu and shortcut features to respond when triggered across various developer platforms, code repositories, or local host environments. We do not read or scrape your page content.

3. Data Security

Since no data is ever transmitted over the internet or sent to our servers, your SQL code (which may contain sensitive database schemas or business logic) remains entirely safe within your local sandbox environment.

4. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page and reflected in the extension's version updates.

5. Contact Us

If you have any questions or suggestions regarding this Privacy Policy, please contact the developer via the official support email or GitHub repository listed in the Chrome Web Store.
