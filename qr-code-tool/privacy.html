隐私政策 (Privacy Policy)

更新日期：2026年5月15日

欢迎使用我们的二维码生成器 Chrome 插件（以下简称“本插件”）。我们非常重视您的隐私，本隐私政策旨在明确向您说明本插件如何处理相关数据。

1. 数据收集与使用

我们承诺：本插件不会收集、存储、记录或传输您的任何个人身份信息、浏览历史或输入的数据。

  • 网页 URL 与选中文本：当您点击插件或使用右键菜单时，插件仅在本地读取当前页面的 URL 或您主动选中的文本，并立即通过本地 JavaScript 库将其转化为二维码。
  • 无网络上传：上述转化过程完全在您的计算机本地浏览器中完成，不会将任何信息上传到我们的服务器或任何第三方服务器。

2. 权限使用说明

为了实现核心功能，本插件可能会向浏览器申请以下必要权限：

  • activeTab：仅用于获取您当前正在浏览的网页地址，以便将其转化为二维码。
  • contextMenus：用于在您右键点击网页文本时，提供“生成二维码”的快捷菜单项。

这些权限仅在您主动触发插件功能时在本地生效，绝不用于后台窃取或监控您的行为。

3. Cookie 与第三方服务

本插件不使用任何 Cookie，不包含任何第三方广告追踪代码，也不集成任何数据分析工具（如 Google Analytics）。

4. 政策变更

如果未来因功能升级需要变更本隐私政策，我们会在插件更新说明中明确提示，并在此页面更新修订日期。但在任何情况下，我们都不会改变“不收集用户隐私数据”的核心原则。
