隐私权政策

最近更新日期：2026年5月20日

本隐私权政策旨在向您说明我们的 Chrome 浏览器扩展程序（以下简称“本插件”）如何处理您的数据。我们非常重视您的隐私，并致力于保护您的个人信息安全。

1. 数据收集与处理原则

我们的核心原则是：您的数据属于您自己，我们绝不收集、不存储、不上传您的任何数据。

  • 本地处理： 本插件的所有数据抓取、表格解析、文件转换（CSV/Excel/JSON）操作全部在您的本地计算机和浏览器中完成。
  • 无远程服务器： 本插件没有配置任何后端服务器，也不会将您浏览的网页内容或提取的表格数据发送给任何第三方。

2. 权限使用说明

为了实现核心的数据导出功能，本插件在您的授权下使用了以下必要权限：

  • activeTab & scripting： 仅在您主动点击插件图标时，用于读取当前活动网页中的 HTML 表格元素。我们不会在后台暗中监控您的浏览行为。
  • storage： 仅用于在您的浏览器本地保存您的个性化配置（如您自定义的抓取行数、首选导出格式），该数据不会同步到外部。
  • 网页访问权限 (<all_urls>)： 允许本插件在您指定的任意合法网页上运行表格提取功能。

3. 数据共享与转让

由于我们不收集任何用户数据，因此我们绝不会、也无法向任何第三方公司、广告商、数据经纪人或个人出售、共享、转让或披露您的数据。

4. Limited Use（有限使用）合规承诺

本插件严格遵守 Chrome Web Store 的开发者计划政策，包括《有限使用政策》。我们对权限的使用完全严格限制在“为用户提供网页表格本地导出”这一单一核心用途内。

5. 政策更新

如果我们对插件的功能进行了重大调整，可能会适时更新本隐私政策。任何更改都会在此页面上公布，且新版本绝不会改变我们“不收集数据”的初衷。

6. 联系我们

如果您对本隐私政策或插件有任何疑问，请通过 Chrome 商店的开发者支持邮箱与我们联系。
