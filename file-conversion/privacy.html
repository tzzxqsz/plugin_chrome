隐私政策 (Privacy Policy)

最近更新日期：2026年5月

本插件（"文件转换助手"）坚决保护您的个人隐私。我们的核心原则是：您的文件属于您，我们永远不会触碰它们。

1. 数据收集与使用

我们的插件遵循“最小化权限”原则：

  • 不收集个人信息： 我们不要求您注册账号，也不收集您的姓名、电子邮件地址或任何身份识别信息。
  • 不收集浏览历史： 我们不会监控您的上网行为或记录您访问的网页。

2. 文件处理逻辑（本地转换）

这是本插件最重要的隐私保证：

  • 所有转换操作（PNG 转 PDF、DOCX 转 PDF 等）均在您的本地计算机上完成。
  • 您的文档、图片或 PDF 不会上传到任何外部服务器。
  • 转换过程通过 JavaScript 在浏览器沙盒内运行，处理完毕后数据即从内存中释放。

3. 权限说明

为了实现功能，本插件可能需要以下权限：

  • Storage： 仅用于存储您的本地偏好设置（如默认导出的 DPI 或语言）。
  • Downloads： 仅用于将转换完成后的结果文件保存到您的电脑下载文件夹。

4. 第三方服务

本插件不包含任何第三方追踪工具、分析插件（如 Google Analytics）或广告平台。我们没有后台系统，因此不存在数据共享。

5. 变更通知

如果未来因功能调整需要更新隐私政策，我们会通过 Chrome 应用商店的更新说明告知用户。继续使用本插件即表示您同意修改后的政策。

6. 联系我们

如果您对隐私处理有任何疑问，可以通过 Chrome 网上应用店的开发者联系渠道与我取得联系。
