隐私政策 (Privacy Policy)

最近更新日期：2026年5月

🛡️ 这是一个“零权限请求”的纯净插件。您的数据永远安全。

1. 核心隐私声明

本插件（"文件转换助手"）的设计初衷是提供一个极简、安全的文件处理方案。我们坚信：您的数据隐私不应作为功能的交换条件。

2. 零权限，更安全

与市面上大多数插件不同，本插件在 manifest.json 中：

  • 未请求 Storage 权限： 我们不存储您的任何数据。
  • 未请求 Downloads 权限： 我们不干扰或监控您的下载列表。
  • 未请求 Host 权限： 我们无法访问您浏览的任何网页内容。

3. 纯本地文件处理

所有的文档转换（DOCX、PDF、PNG 互转）逻辑均通过 JavaScript 在您的浏览器本地内存中即时运行：

  • 文件无需上传至远程服务器。
  • 转换过程中不产生任何云端缓存。
  • 一旦您关闭转换窗口，内存中的相关数据将立即释放。

4. 数据收集与追踪

我们承诺：

  • 不收集任何形式的个人身份信息。
  • 不使用任何第三方追踪工具（如 Google Analytics、Cookies）。
  • 不展示任何形式的广告。

5. 如何保存转换后的文件？

由于我们未申请下载权限，文件转换完成后，插件会调用浏览器原生的下载机制由用户手动确认保存。这确保了整个过程完全在您的掌控之下。

6. 联系我们

如果您对本政策有任何疑问，请通过 Chrome 网上应用店的开发者反馈渠道联系我们。
