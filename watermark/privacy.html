隐私政策 (Privacy Policy)

更新日期：2026年6月

本隐私政策旨在向您说明我们的 Chrome 浏览器扩展程序（以下简称“本插件”）如何处理您的数据。我们非常重视您的隐私，并承诺保护您的个人信息。

核心核心： 本插件是一款【全本地运行】的工具。我们不会收集、存储、传输或共享您的任何个人数据、网页浏览历史或处理的图片。

1. 数据收集与使用

  • 图片数据： 当您上传本地图片或选择网页图片添加水印时，所有图片处理（Canvas 绘制）均在您的本地浏览器中完成。图片绝对不会被上传到任何外部服务器。
  • 配置数据： 我们使用 Chrome 提供的 storage API 在本地保存您自定义的水印样式（如文本内容、颜色、大小）。这些数据仅保存在您的本地设备上。
  • 个人信息： 我们不要求您注册账户，不收集您的姓名、电子邮箱、IP 地址或任何其他个人身份信息。

2. 权限使用说明

  • storage：仅用于在本地记住您常用的水印设置，避免重复输入。
  • host_permissions (<all_urls>)：仅用于允许您在浏览任意网页时，能够提取该网页上的图片并将其导入到水印编辑器中。

3. 第三方服务与数据传输

本插件不包含任何第三方统计代码、广告联盟或追踪器。我们绝不会将任何数据出售或披露给第三方。

4. 政策变更

如果我们未来对插件功能进行升级并涉及隐私政策的变更，我们会在此页面更新本政策。建议您定期查看。

5. 联系我们

如果您对本隐私政策或本插件有任何疑问，请通过 Chrome Web Store 的开发者联系渠道与我们取得联系。
