隐私政策 (Privacy Policy)

最近更新日期：2026年6月2日

本隐私政策适用于您使用“代码比对工具”（Code Diff Tool）Chrome 浏览器扩展程序（以下简称“本插件”）。我们高度重视您的隐私，并承诺保护您的代码安全。

1. 数据收集与使用

我们不收集、不存储、不传输任何代码或个人数据。

  • 本地比对： 当您输入代码或选择两个本地文件夹进行差异比对时，所有文本处理和比对算法均完全在您的本地浏览器内存中运行。
  • 无网络通信： 本插件未设计任何后端服务器，不包含任何网络请求逻辑（如 fetch 或 XMLHttpRequest），绝对无法将您的代码数据上传到互联网。

2. 权限声明

本插件仅依赖浏览器标准的 HTML5 File API 处理您主动选择的文件。我们不请求任何可以追踪您浏览历史、敏感数据或网页内容的敏感权限。

3. 变更通知

如果未来本插件的隐私政策有所变更，我们将在本页面进行更新。但在任何情况下，我们绝不改变“不收集用户源码”的核心隐私安全底线。

4. 联系我们

如果您对本隐私政策有任何疑问，请通过 Chrome 应用商店的开发者反馈渠道与我们取得联系。

Privacy Policy

Last updated: June 2, 2026

This Privacy Policy applies to your use of the "Code Diff Tool" Chrome extension (the "Extension"). We highly value your privacy and are committed to protecting your source code.

1. Data Collection and Usage

We do not collect, store, or transmit any source code or personal data.

  • Local Processing: When you paste code or select local directories for diffing, all text analysis and comparison algorithms are executed entirely within your local browser environment.
  • No Network Communication: The Extension has no backend server and includes no tracking or network request capabilities. Your files will never be uploaded to the internet.

2. Permissions

The Extension relies solely on standard HTML5 File APIs to process user-selected text. No high-risk permissions tracking your browsing habits or web data are requested.

3. Contact Us

If you have any questions regarding this Privacy Policy, please contact us via the Chrome Web Store support/developer channel.
