Developer Toolkit 隐私政策

最近更新日期：2026 年 6 月 3 日

核心隐私承诺：

本插件（Developer Toolkit / DevTools Toolbox）坚决捍卫用户的数据隐私。我们的核心宗旨是：所有数据完全留在本地，绝不上传云端。 本插件不设任何后端服务器，亦不包含任何第三方追踪或统计 SDK。

1. 数据收集与处理声明

当您使用本插件集成的一系列开发辅助工具（包括但不限于：SQL 格式化、JWT 解码、正则表达式测试、UUID 生成、时间戳转换、代码比对、代码行数计算、Markdown 预览等）时，您的数据流向声明如下：

  • 完全本地化运算： 您在各功能输入框中粘贴、输入的任何敏感代码、文本、Token 或数据库语句，均由您的浏览器在本地实时处理，完全不经过任何远程服务器传输。
  • 不收集个人隐私： 我们不会收集您的姓名、电子邮箱、设备信息、IP 地址或任何形式的网页浏览历史（Browsing History）。

2. 权限声明及最小权限原则

为了保障各项独立工具合并后的正常运转，本插件在 manifest.json 中声明了以下必要权限。我们严格将权限用途限制在以下范围内：

  • clipboardWrite（剪贴板写入权限）： 仅在您主动点击工具内的“复制”按钮时触发。用于将格式化后的 SQL、新生成的 UUID、或解码后的 JWT 负载等结果安全地复制到您的系统剪贴板中。
  • storage（本地存储权限）： 仅用于在您的浏览器本地缓存用户的偏好设置（如代码比对的主题颜色、常用配置）以及部分工具的临时文本占位，以便您下次打开插件时能够无缝继续工作。
  • contextMenus（右键菜单权限）： 用于在浏览器中注册快捷右键菜单（例如：右键一键格式化选中的 SQL 语句或一键转换选中时间戳），减少切换插件的步骤。
  • activeTab（当前标签页响应权限）： 仅在您通过点击插件图标或使用右键菜单时临时激活，用于使插件安全地响应当前网页的上下文，不用于任何后台常驻网页监听。

3. 外部服务与第三方共享

本插件是一款纯净的本地效率工具箱：

  • 不包含任何形式的第三方广告、联盟链接或内购组件。
  • 绝不向任何第三方机构、公司或个人出售、共享或披露您的任何数据。
  • 后续计划更新的功能（如 JSON Formatter、Base64 编解码、Hash 生成器等）将继续贯彻 100% 本地封装与运行的原则。

4. 政策变更与迭代

随着插件功能的持续丰富与版本迭代，本隐私政策可能会进行细节更新。然而，无论插件如何升级，“不收集、不传输、不上传用户任何输入数据”的最高隐私准则将永远保持不变。如果发生重大变更，我们将在插件更新日志中显著向用户说明。

5. 联系与反馈

如果您对本隐私政策或本插件的数据处理方式有任何疑问、意见或建议，请随时通过 Chrome 网上应用店（Chrome Web Store）的“支持”与“反馈”板块与我们取得联系。
