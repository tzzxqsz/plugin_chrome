Privacy Policy / 隐私政策

Last Updated: June 2026

1. Data Collection and Usage / 数据收集与使用

Our Chrome extension is designed with a privacy-first approach. We DO NOT collect, store, or transmit any personally identifiable information (PII), browsing history, or user data.

本 Chrome 插件在设计上确立了隐私至上的原则。我们不会收集、存储或传输任何个人身份信息（PII）、浏览历史记录或用户数据。

2. Network Requests & API Permissions / 网络请求与权限说明

The extension requests host permissions for https://github.9191ta.com/* and https://rank.9191ta.com/* solely to fetch publicly available trending data (e.g., GitHub trending repositories, Hacker News top stories). These requests are completely anonymous and do not contain any user cookies, tokens, or identifiers.

本插件请求 https://github.9191ta.com/* 和 https://rank.9191ta.com/* 的主机权限，仅用于获取公开的趋势数据（例如 GitHub 热门项目、Hacker News 热点）。这些请求完全匿名，不包含任何用户 Cookie、Token 或识别码。

3. Data Security / 数据安全

All communication between the extension and our servers is encrypted using standard HTTPS protocol to ensure data integrity and prevent tampering.

插件与我们服务器之间的所有通信均使用标准的 HTTPS 协议进行加密，以确保数据完整性并防止被篡改。

4. Third-Party Services / 第三方服务

We do not integrate any third-party tracking, analytics, or advertising SDKs within the extension.

我们没有在插件中集成任何第三方追踪、数据分析或广告 SDK。

5. Changes to This Policy / 政策变更

We may update our Privacy Policy from time to time. Any changes will be posted on this page.

我们可能会不时更新隐私政策。任何变更都将在此页面上公布。

6. Contact Us / 联系我们

If you have any questions about this Privacy Policy, please contact us via our official support channels.

如果您对本隐私政策有任何疑问，请通过官方支持渠道与我们联系。
