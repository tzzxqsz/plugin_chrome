隐私政策 (Privacy Policy)

本隐私政策阐明了我们的 Chrome 扩展程序（以下简称“本插件”）如何处理相关信息。我们极度重视用户的隐私，并严格遵守 Chrome Web Store 开发者服务条款。

1. 数据收集与使用

我们承诺不收集任何个人隐私数据。

  • 浏览历史：本插件不会读取、记录、追踪或存储您的任何浏览器历史记录或访问过的网页。
  • 个人信息：本插件不需要您注册账号，不收集您的姓名、电子邮箱、设备信息或 IP 地址。
  • 输入数据：本插件在运行期间不输入、不传输任何用户本地生成的数据。

2. 网络请求与权限声明

本插件在 manifest.json 中仅声明了必要的 host_permissions（即 https://rank.9191ta.com/*）。该权限的唯一用途是向我们的服务器发出网络请求，以获取最新的全球 Top 100 网站排行榜 JSON 数据。

在与服务器通信的过程中，本插件不会附带任何可识别用户身份的特征码或 Cookie。

3. 数据安全

由于本插件不在服务器端存储、处理或传输任何用户数据，因此不存在用户数据泄露的风险。所有的网络通信均通过安全的 HTTPS 加密协议进行。

4. 政策变更

如果我们对本插件的功能进行升级并涉及隐私政策的调整，我们会提前更新本页面。继续使用本插件即表示您同意本隐私政策。
