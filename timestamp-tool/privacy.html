隐私政策 (Privacy Policy)

更新日期：2026年5月

本隐私政策旨在向您说明本 Chrome 扩展程序（以下简称“本插件”）如何处理您的数据。我们非常重视您的隐私，本插件的设计核心原则是：不收集、不传输、不共享您的任何个人数据。

核心承诺： 本插件是一款纯客户端运行的工具。所有时间的计算、转换以及历史记录的存储，全部在您的本地浏览器中完成。我们没有任何外部服务器，绝对不会将您的数据上传到网络。

1. 权限使用与数据收集说明

  • Context Menus (右键菜单权限)： 仅用于在系统右键菜单中生成“转换时间戳”快捷项，不读取任何行为数据。
  • Storage (本地存储权限)： 仅用于在您的本地浏览器中（通过 chrome.storage.local）保存您的历史转换记录和设置项（如时区、时间格式）。这些数据永远不会离开您的设备。
  • 网页内容读取 (Content Scripts)： 仅用于在您主动“划词选中”或“双击”网页中的时间戳数字时，在本地为您计算并弹出浮窗显示时间。本插件不会读取、解析或记录您浏览的任何其他网页内容或隐私数据。

2. 第三方服务

本插件不集成任何第三方统计、分析 SDK，也不包含任何广告追踪器。整个运行过程完全独立、干净。

3. 隐私政策的变更

如果未来因为功能迭代需要调整权限，我们会在新版本发布时同步更新此隐私政策。但“本地运行、保护隐私”的底线永远不会改变。

4. 联系我们

如果您对本隐私政策或本插件有任何疑问，请通过 商店的开发者反馈渠道与我们联系。
